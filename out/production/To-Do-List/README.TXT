Project: To-Do-List Application
Author: Sudha Bommakanti

This project is a simple java application for to-do-lists.
It has a text user interface where the user has the following options.
1)  create a new task and store or add it to this to do lists.
2)  view all the tasks
3)  Edit any task
4)  save and quit


To start this application, the user should create an instance for the
main class and call its "list" method.
